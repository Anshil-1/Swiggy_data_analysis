
# Swiggy-Style Food Delivery Data Analysis (PostgreSQL)

End-to-end PostgreSQL project covering data cleaning, normalization,
advanced SQL queries, CTEs, window functions, stored procedures,
and performance optimization.

## Features
- Large realistic dataset
- 20+ SQL analytical queries
- Indexing & EXPLAIN ANALYZE
- Business insights (top customers, slow riders)

## How to Run
1. Run schema/create_tables.sql
2. Import CSV files from data/
3. Execute queries from queries/
4. Run procedures & performance scripts
